# React JSX Lab

## Overview
This lab demonstrates how to create React components using JSX syntax. It covers functional components, rendering components to the DOM, using dynamic data, rendering lists, and component composition.

## Features
- Functional React components
- JSX syntax
- Dynamic data rendering
- List rendering using map()
- Reusable components

## Challenges Encountered
One challenge encountered was configuring Babel correctly for JSX transpilation. Another was ensuring the project structure and imports were properly organized.

## How to Run
1. Install dependencies:
   npm install

2. Build the project:
   npm run build

3. Open public/index.html in the browser.
